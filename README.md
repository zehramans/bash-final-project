loggex is a simple tool to analyze apache and ssh log files, print out the the ip addresses that occur the most and detect potential bruteforce or sql injection attacks.
USE:
-f = specify log file name
-F = specify log file format
-d = what should be detected (for now only brute and injection options are available)
